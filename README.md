# Chess

Small local chess project. Run the game with:

```bash
python3 main.py
```

Steps to publish this repo to GitHub:

1. Initialize git locally: `git init`
2. Add files and commit: `git add . && git commit -m "Initial commit"`
3. Create a GitHub repo and add remote (using `gh` or create on GitHub web)
4. Push: `git push -u origin main`
